*easy_ecc

此题曲线为蒙哥马利形式
$by^2$ = $x^3+ax^2+x$ mod p
将其映射成维尔斯特拉斯曲线
$y^2$ = $x^3+Ax+B$ mod p
映射规则如下:
A = $\frac{3-{a^2}}{3b^2}$
B = $\frac{2a^3-9a}{27b^3}$
其点映射为:
x = $\frac{3x+a}{3b}$ y = $\frac{y}{b}$

映射完后发现满足$4a^3+27b^2$ = 0 mod p
也就是说曲线是奇异，可以将其转为模p的数域上
